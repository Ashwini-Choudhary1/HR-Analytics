# HR Analytics: IBM Employee Attrition & Retention Intelligence 👥

**Predictive Attrition Risk Modeling**  
*An end-to-end Python, Machine Learning, SHAP, and Tableau analysis of 1,470 IBM employee records across 35 features to predict turnover and design retention strategies.*

---

## 📁 Repository Structure

- **`Data/`**: Dedicated folder for the IBM Employee Attrition dataset (e.g., CSV files and cleaned feature-engineered datasets).
- **`Analysis Notebook/`**: Dedicated folder for Jupyter Notebooks (`.ipynb`) containing PACE-structured EDA, Random Forest & XGBoost classification models, and SHAP explainability analyses.
